Send Events Using C#
===============

Describes how to send an event through the Whatsnexx.TicketBus from a C# project

Getting Started
------------------
The whatsnexx [ticketbus](https://github.com/whatsnexx/Whatsnexx/wiki/TicketBus) handles REST and WCF Web Service request to send events.
